# 머슬킵 (musclekeep) — GLP-1 근육 지키기 코치 앱

> 위고비·마운자로 복용 중 **근육을 지키며 감량**하도록, 매일 단백질·저항운동·주사일을 챙겨주는 AI 코치. (PWA · 순수 정적, 빌드 없음)

## 🔗 라이브
- 앱: https://togetherwell1-cmd.github.io/musclekeep-app/
- 리드마그넷(자가진단 계산기): https://togetherwell1-cmd.github.io/glp1-muscle-check/

## ✅ 완료 (Week 1~3)
- **W1** 온보딩(약·기간·목표·나이·체중·운동·단백질·감량속도) → 프로필 저장 → 하루 단백질 목표·근손실 위험도·오늘 할 일 자동 계산. 홈 대시보드 + 탭바. PWA.
- **W2** 기록: 단백질 원터치 로깅·체중·근력·부작용 + 홈 진행바 실데이터 + 연속기록(streak) + 최근 7일.
- **W3** 루틴: "오늘 뭐 먹지"(남은 단백질 채우는 한식 추천) + 저항운동 루틴 + **추천→원터치 기록** 루프. 현재 규칙기반 엔진(키/서버 불필요), LLM은 `aiRecommend()` 훅에 서버리스로 확장 예정.
- **W4** 주사일: 주기(주1회/매일)·요일·시간 등록 → 홈에 **다음 주사 D-day** + "오늘 주사일" 알림 + **주사 완료 기록**. 서버 없이 실제 알림: **폰 캘린더에 반복 일정(.ics) 추가** + 브라우저 Notification.

## 🗓 다음 로드맵
- **Week 5** 추세 그래프(체중·단백질·근력 유지) + 주간 리포트
- **Week 6** 구독 결제 연동 + 파일럿 오픈

## 실행/개발
- 정적 파일이라 서버 불필요. 로컬 확인: `python3 -m http.server` 후 브라우저.
  (서비스워커는 `http://localhost` 또는 https에서만 동작)
- 배포: GitHub Pages(main 브랜치 / 루트). push 하면 자동 반영.
